# 模板选择规则

已知模板：
- sci论文-标题不编号.docx
- sci论文-标题编号.docx
- 标题不编号-列表第二行缩进.docx
- 标题不编号-列表第二行顶格.docx
- 标题编号-列表第二行缩进.docx
- 标题编号-列表第二行顶格.docx

## 选择维度

优先按这几个维度判断：

1. 是否是 SCI 论文场景
2. 标题是否需要编号
3. 列表第二行是否需要缩进
4. 列表第二行是否需要顶格

## 建议映射

### SCI 论文
- 标题不编号 → `template_sci论文-标题不编号.docx`
- 标题编号 → `template_sci论文-标题编号.docx`

### 普通文档
- 标题不编号 + 列表第二行缩进 → `template_标题不编号-列表第二行缩进.docx`
- 标题不编号 + 列表第二行顶格 → `template_标题不编号-列表第二行顶格.docx`
- 标题编号 + 列表第二行缩进 → `template_标题编号-列表第二行缩进.docx`
- 标题编号 + 列表第二行顶格 → `template_标题编号-列表第二行顶格.docx`

## 注意事项

- 模板决定的是 Word 样式基准，Lua 过滤器主要负责补强 Pandoc 默认能力。
- 图片编号是否自动添加，不由 reference-doc 单独决定，还和是否启用 `image-title-to-caption-add-number.lua` 有关。
- 如果用户明确指定模板文件名，优先尊重用户指定。
